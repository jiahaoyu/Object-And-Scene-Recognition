## 3.相关技术学习路线

首先是python的语言，虽然说不熟悉也能写东西，但是并不能写出高效的代码，因此尽量熟悉它。

较为重点的内容在算法方面，也就是机器学习下的神经网络和深度学习方面的算法。至于神经网络，基础概念包括对神经元的理解以及线性可分的神经网络概念，重点还在于理解它的“层”的概念。

其次是关于框架，Tensorflow作为一个开发框架，我们需要去了解他的运作方式，以及它在各种神经网络模型下的简单应用。
